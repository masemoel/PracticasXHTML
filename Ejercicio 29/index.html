Insertando imágenes en documentos HTML

Las primeras páginas web no tenían imágenes y se limitaban a mostrar textos porque el ancho de banda era muy pequeño. En la actualidad, la mayoría de las páginas de Internet están ilustradas con imágenes. El diseño de las páginas web en HTML permite insertar imágenes fácilmente.

Esta es la primera imagen que inserto en mi página web utilizando una orden HTML.

En este caso se trata de una foto que he conseguido en una página web donde hay muchas imágenes para descargar.

He colocado la foto en la parte izquierda de la pantalla, de forma que este texto queda situado en la parte derecha.
